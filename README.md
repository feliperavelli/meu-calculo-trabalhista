# Meu Cálculo Trabalhista

MVP funcional em Next.js para uma calculadora brasileira de rescisão trabalhista online.

## Rodar localmente

```bash
npm install
npm run dev
```

Depois acesse `http://localhost:3000`.

## Scripts

- `npm run dev`: ambiente local de desenvolvimento.
- `npm run build`: build de produção.
- `npm run start`: servidor de produção após o build.
- `npm run typecheck`: validação TypeScript.

## Deploy na Vercel

1. Suba o projeto para um repositório Git.
2. Importe o repositório na Vercel.
3. Use o preset de Next.js.
4. Configure `NEXT_PUBLIC_SITE_URL` com o domínio final.
5. Publique.
